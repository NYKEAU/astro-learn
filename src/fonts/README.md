# Polices pour Astro Learn

Ce dossier contient les polices utilisées dans l'application Astro Learn.

## Polices incluses

- JetBrains Mono (Regular, Bold)

## Instructions

Vous devez télécharger les fichiers de police suivants et les placer dans ce dossier :

1. JetBrainsMono-Regular.woff2
2. JetBrainsMono-Bold.woff2

Vous pouvez télécharger ces polices depuis le site officiel de JetBrains Mono :
https://www.jetbrains.com/lp/mono/

Alternativement, vous pouvez modifier le fichier `layout.jsx` pour utiliser Google Fonts pour toutes les polices.
